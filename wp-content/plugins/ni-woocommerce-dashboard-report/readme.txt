Ni WooCommerce Dashboard Sales Report provide the option to show the order status summary and recent order status on dashboard.
 
== Description ==

Ni WooCommerce Dashboard Sales Report provide the quick information about store sales order on WordPress dashboard, this report provide the summary of each order status with their order total  and order count. This report also show the recent sales order on the dashboard with order details like order total, order status, order email etc.

Key Features

* Display report on WordPress dashboard
* Show five recent order with details information like order ID, Order Status, order total, etc.
* Show the all summary of all order status with order status count and order total
* Ni WooCommerce Sales By Month Report show the total gross sales and total net sales of last 6 month.

1. Use the WordPress plugin installer or upload files to WordPress plugin directory.
2. Activate plugin from admin panel.
3. Go to wordpress dashboard end find Ni WooCommerce Sales Order Status and Ni WooCommerce Recent Order.
